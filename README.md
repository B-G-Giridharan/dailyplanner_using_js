# 📅 Daily Planner (Vanilla JS)

A clean, efficient, and interactive web-based **Daily Planner** designed to help users organize their tasks and manage their time effectively. This project focuses on seamless DOM manipulation and data persistence using browser storage.

---

## 🚀 Live Demo
Check out the live application here: ( https://b-g-giridharan.github.io/dailyplanner_using_js/ )

---

## ✨ Features

- **Task Management:** Easily add, view, and delete daily tasks.
- **Time-Slot Organization:** Organize your day hour-by-hour or by specific categories.
- **Data Persistence:** Uses `localStorage` so your tasks remain saved even after closing the browser or refreshing the page.
- **Responsive Design:** Fully functional on desktops, tablets, and mobile devices.
- **Interactive UI:** Real-time updates without page reloads using Vanilla JavaScript.

---

## 🛠️ Tech Stack

- **HTML5:** Semantic structure for the planner layout.
- **CSS3:** Custom styling with a focus on clean typography and responsive grids.
- **JavaScript (ES6+):** Logic for task handling, event listeners, and local storage integration.
